# 更新日志

所有重要更改都会记录在此。

## [1.7.0] - 2026-05-30

### 重要变更
- **彻底删除惩罚任务计时器**：惩罚不再有任何读秒或自动结束机制。只剩受罚玩家主动点击「我已完成惩罚」按钮，或在座玩家断线/离开时才清除惩罚面板。
- **移除出拳时限设置**：创建房间时已完全删除「每局出拳时限」选项（10秒/15秒/20秒/30秒等），现在默认永久无限制。
- **惩罚风格选择器重构**：从下拉框改为**卡片式选择**（共6个选项），每个选项都带有清晰的中文描述。
  - 无惩罚、幻想/角色扮演、轻度SM惩罚、中度SM惩罚、重度SM惩罚、纹身惩罚
  - 卡片排版针对手机做了优化，触控区域大、文字清晰、视觉区分度高。
- **创建房间弹窗移动端体验大幅提升**：移除了时限那一整块后，界面更简洁干净，滚动压力更小，尤其适合手机操作。

### 改进
- 惩罚面板彻底移除倒计时相关 DOM 和逻辑，界面更干净。
- 桌子内 round 提示也同步更新为“无出拳时限（默认）”。

## [1.6.0] - 2026-05-30

### 新增功能
- **轻度SM惩罚扩充至50条**：全部重写为更涩、更详细的轻度SM任务（身体书写、M字开腿掰穴/蛋蛋展示、自扇下体+报数、深喉模仿+自摸、强制自述羞耻语等）。
- **纹身贴任务系统**：新增20条固定示例 + 完整随机生成器。
  - 支持40+个可写部位（从额头、耳垂、乳头、龟头、阴蒂、蛋蛋、尾椎、脚心到腹股沟等全部覆盖）。
  - 30+条羞耻文字（「公共肉便器」「主人精液罐」「zako专用」「请内射」「免费使用」「打屁股专用」等）。
  - 随机持续2-5分钟 + 额外羞耻条件（自拍、保持姿势、边贴边念）。
- **性别系统完整版**：创建/修改名字时可选♂️男性 / ♀️女性 / 🌸男娘。
  - 所有名字显示处自动带性别图标（玩家卡、聊天、排行榜、惩罚历史、结果记录等）。
  - 顶部昵称栏新增性别下拉，可随时修改性别并同步全桌。
  - 服务端完整支持性别字段持久化、广播、断线恢复。

### 改进
- 惩罚风格支持新增「纹身贴」选项（随机部位+文字，极致羞耻）。
- 清理了服务端重复的惩罚代码，全部统一走独立 punishment.js 模块。
- 性别数据在 join、座位、观众、断线重连全链路同步。

### 修复
- 修复了旧代码中残留的重复 FANTASY/LIGHT 列表导致的语法问题。

## [1.5.0] - 2026-05-29

### 新增功能
- **断线保护**：玩家断线后座位保留45秒，支持重连恢复。断线期间实时显示倒计时。
- **惩罚中的玩家断线处理**：如果正在受惩罚的玩家断线，会暂停惩罚倒计时；超时未返回则自动取消惩罚。
- **房间设置增强**：
  - 支持多种游戏模式（无限制、3局2胜、5局3胜、先到5胜）
  - 创建房间时可选择密码房（私人房间）
- **惩罚风格**：使用下拉选择框，方便后续扩展更多惩罚类型。
- **快速消息系统**（带防刷屏）：点击玩家名字可快速发送“快点！”“你好弱”“zako”等消息，带有客户端+服务端双重冷却限制。
- **观众增强**：
  - 观众可直接点击空座位快速上桌
  - 大厅和桌子内清晰显示在座玩家与观众列表
- **全局排行榜**：大厅显示胜场、胜率、被罚次数等排行。
- **个人惩罚历史**：大厅显示玩家自己的最近惩罚记录。
- **实时倒计时**：惩罚30秒和断线45秒保护均采用服务器权威时间，通过WebSocket实时推送。
- **大量移动端优化**：响应式布局、按钮大小自适应、聊天抽屉、避免按钮溢出等。

### 改进
- 重置比分时会同步清除当前惩罚状态。
- 密码房加入时会弹出密码输入。
- 优化了聊天输入框在窄屏下的布局（发送按钮不再溢出）。
- 提升了惩罚和断线状态的同步稳定性。

### 修复
- 修复了移动端标题显示问题。
- 修复了惩罚倒计时在1秒时卡住的问题。
- 修复了观众离开错误影响惩罚状态的bug。
- 修复了发送按钮在某些情况下点不到的问题。
